# Watering controller

## Uruchomienie:
`java WateringFuzzy <plik_fcl> <stopien nawadniania> <temperatura> <wilgotnosc>`

## Wejście
Stopień nawadniania: (0 .. 10)
Temperatura: (0 .. 50)
Wilgotność: (0 .. 100)

## Wyjście
Zmiana stopnia nawadniania: (0 .. 100)

## Przykład
`java WateringFuzzy watering_controller.fcl 3 25 50`
